Provenance ensemblec back to ToC or Class ToC

IRI: https://w3id.org/okn/o/preo#ProvenanceEnsemble

A special type of activity representing an ensemble of executions submitted to a workflow system. The ensemble may be on parameters (e.g., range variation), inputs (e.g., varying several files) or even the software executed (in case multiple software steps are to be executed with the same inputs).
